Skills Showcase 管理

本页只编辑 site-config.json -> sections.skills.title、sections.skills.description 和 sections.skills.showcase；不会编辑技术栈蜘蛛网节点（sections.skills.items）。

下载后上线流程

  1. 用下载的 site-config.json 替换 portal/site/assets/site-config.json。
  2. 执行 git add portal/site/assets/site-config.json，然后 Git 提交并推送。
  3. 服务器进入仓库后执行 git pull。
  4. 执行 docker compose up -d --build 完成重建。

点击下载前会再次确认：导出会自动修正 sequence、tags、metric、featured、id、ctaLabel，不是原样导出当前表单值。

本页面不会直接写服务器文件，也不会修改技术栈蜘蛛网、HireMate、Caddyfile 或 shared_gateway。
